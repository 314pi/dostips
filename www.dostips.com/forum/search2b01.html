Re: Create folders in ascending order

Something like that?

Code: Select all

@echo off
setlocal EnableDelayedExpansion
set /p count=How many folders?
for /L %%i in (1, 1, %count%) do (
 set "format=000000%%i"
 md folder!format:~-3!
)
goto :eof
