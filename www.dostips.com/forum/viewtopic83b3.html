Posted: 05 Apr 2011 22:46
by DosItHelp
The double quotes are trouble.
5th character of string1 is not a double quote.

this should work:
set string1="test"ing
set string2="test"

or this:
set "string1=testing"
set "string2=test"

or this:
set string1=testing
set string2=test

or even this
set "string1="test"ing"
set "string2="test""

but unbalanced quotes however will cause trouble in the IF statement:
set "string1="testing"
set "string2="test"
