Re: Macros with parameters appended

The \n value used to create multi-line macros can be greatly simplified as ^<LF> instead of ^<LF><LF>^.

Code: Select all

(set \n=^^^
%= This creates an escaped Line Feed - DO NOT ALTER =%
)
