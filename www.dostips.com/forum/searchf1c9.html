Re: Using the strLen function with my script. Please Help

Try this:

Code: Select all

@echo off
setlocal enabledelayedExpansion
for /r . %%g in (*.*) do (
      call :strLen %%g var1
      echo Pfad ist !var1! Zeichen lang
      echo %%g
)      
set cnt=0
for %%A in (*) do set /a cnt+=1
set /A cnt=cnt-1
echo Dateien im Verzeichnis = %cnt%
PAUSE
exit /b
