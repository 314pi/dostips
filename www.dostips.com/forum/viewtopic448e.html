#5 Post by ShadowThief » 30 Mar 2018 18:34

It should be as simple as

Code: Select all

@echo off
setlocal enabledelayedexpansion

for /f "delims=" %%A in (test1.csv) do (
	set "tab_line=%%A"
	set "comma_line=!tab_line:	=,!"
	echo !comma_line! >>test.csv
)
