Re: how to backup a text file every hour in windows 7?

Posted: 21 Jul 2018 00:13
by Squashman

Code: Select all

@echo off
FOR /F "tokens=2 delims==.-" %%G IN ('wmic os get localdatetime /value') DO SET "dt=%%G"
set "dt=%dt:~0,12%"
copy "scratch.txt" "scratch-%dt%.txt"
