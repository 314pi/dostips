This might help you (untested):

Code: Select all

:listPar                        
for /F "usebackq delims=" %%? in (`"%~1|findstr "^^Type""`) do (
   set "result=%%~?"
)
set "result2=%result:*:=%"
set "result2=%result2: =%"

:: debug output:
set result

goto :eof
