Re: How to replace space in file names

How do SETLOCAL and ENABLEDELAYEDEXPANSION work?

Code: Select all

@echo off
Set "Pattern= "
Set "Replace=_"
Setlocal EnableDelayedExpansion
For /f "delims=" %%a in ('dir /b /s "*.txt"') Do (
Set "File=%%~nxa"
echo Ren "%%a" "!File:%Pattern%=%Replace%!"
)
pause
