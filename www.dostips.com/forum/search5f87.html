Re: How to call the oldest batch file from another batch fil

Maybe this piece of code will help you.

Code: Select all

pushd "C/app/outbound"
for /f "delims=" %%a in ('dir /b /o-d *.bat') do set "oldBat=%%~fa"
popd
echo oldBat == %oldBat%
pause

(untested)
