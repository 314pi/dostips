I suppose you want to create a path "c:\temp\myPath\mySrc" if it is not exists.

you can simple use

Code: Select all

if the path, it will result with an error, but that can be ignored.

Hi,

reference at FOR /?

splitPath.bat

Code: Select all

echo Path=%~dp1
echo filename=%~nx1


Use it like

Code: Select all

splitPath "c:\tmp\tmp1 add\file 22.exxt"
